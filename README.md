# LR / SLR Terminal Challenge

Interactive educational challenge inspired by compiler construction and bottom-up parsing techniques used in LR and SLR parsers.

Live Demo:
https://kenpachiryu.github.io/lr-slr-terminal-challenge/

--------------------------------------------------

## Overview

This project was developed as an interactive learning activity for the Compilers course.

The application simulates a futuristic terminal where students solve challenges related to:

- LR(0) parsing
- SLR(1) parsing
- Shift-Reduce parsing
- DFA parser behavior
- FOLLOW sets
- Stack operations
- Compiler theory concepts

The experience combines:
- multiple choice questions
- parser simulations
- drag & drop logic
- glossary matching
- custom terminal-style word search

--------------------------------------------------

## Features

- Cyberpunk / terminal-inspired interface
- Fully client-side application
- Randomized activities and questions
- Drag & Drop interactions using SortableJS
- Persistent activity state while navigating
- Real-time timer
- Automatic score calculation
- Team registration system
- Procedural terminal background animation
- Responsive layout

--------------------------------------------------

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- SortableJS

--------------------------------------------------

## Educational Purpose

The objective of this project is to reinforce theoretical concepts of syntactic analysis through interactive problem solving.

Topics included:
- Shift / Reduce operations
- Viable prefixes
- LR items
- Parser stack behavior
- DFA transitions
- FOLLOW sets
- Shift-Reduce conflicts
- Reduce-Reduce conflicts
- Grammar augmentation
